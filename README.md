## 项目介绍

该项目是我的本科毕业设计项目的前端部分，是一个家庭资产管理系统，主要功能包括资产管理、账单管理、预算管理、数据统计等。

## 项目技术栈

- Vue3
- Vue Router
- Pinia
- Element Plus
- ECharts
- Axios
- Webpack5
- Scss

## 项目运行

```bash
# 安装依赖
npm install

# 启动项目
npm run build
```
